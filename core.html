GestureWorks Core

The Most Powerful Authoring
Solution for Multitouch Ever Built

GestureWorks Core is an incredibly efficient and powerful authoring solution and touch point cluster analysis system for multitouch. GestureWorks Core includes a library of over 300 pre-built gestures and includes support for the world’s first markup language for multitouch, the Gesture Markup Language (GML).

Multi-Language Support &
A Variety of Frameworks

GestureWorks Core for C++, .NET(C#, VB), Java, Python, and JavaScript(HTML5) allows you to author in the language of your choice (ActionScript developers can use Open Exhibits). GestureWorks Core provides standardized development and support for GML across multiple languages. We have detailed tutorial examples showing GestureWorks Core with Cinder, openFrameworks, Unity 3D, XNA, Kivy, Java 2D, Processing, and Panda3D.

Works on Hundreds of
Multitouch Devices

GestureWorks Core works on virtually any Windows 8, Windows 7, or TUIO compatible touch-enabled device or screen. In addition, it supports the full spectrum of large-scale multitouch screen technologies such as:

  • Infrared-vision based tables (Surface1, MultiTaction, PixelSense, SUR40)
  • IR- frame based (PQ labs, ZaagTech, ShadowSense)
  • Projected Capacitive (3M Touch System, Ideum, ELO IntelliTouch Plus, Perceptive Pixel)

The efficiency of GestureWorks Core means your projects will have extraordinary performance regardless of the hardware device.

Windows 8 &
Windows 7 Compatible

GestureWorks Core is streamlined to take
advantage of Windows 8 and Windows 7
native touch events. GestureWorks-built
applications can run on any multitouch or
touch-enabled hardware that supports
Windows 8 or Windows 7 or that uses the
TUIO or other touch event protocols.

Revolutionary Gesture-Based
Authoring

The GestureWorks Core package comes complete with definitions for scores of pre-built gestures and possibilities for virtually any number of gestures. GestureWorks Core has support for the Gesture Markup Language (GML), the world’s first markup language for HCI. All of the gestures are fully editable and extensible. Support for physical properties like inertia filters, noise filters and others are built in. GML was invented by GestureWorks.

Universal HCI Engine
Coming Soon!

GestureWorks Core is the product of years of development and its architecture has been designed from the ground up. Unlike other authoring environments, GestureWorks Core has been developed to be a universal HCI engine. Along with touch screens and a variety of motion recognition devices, future plans for GestureWorks Core include the use of accelerometers, wearables, eye-tracking, and other HCI devices. These developments are underway and a release is planned for late in 2014.
